Navigation bar
The navigation bar which consists of the 4 main pages was created to make it easier to browse through the website rather than just using the links.
Accordion tabs
The accordion tabs are created to contain the answers for each of the questions listed for higher readability and a neater appearance for user.
Display appropriate messages for mandatory text fields when hover
A brief message is displayed underneath mandatory textfields according to their requirements when hover or clicked on such as status code and status description at post status page and search status page, to ensure correct input from user. This also saves a lot of time and effort as user can avoid submitting wrong input and has to refill the whole form again.

I had issues with storing data into database, particularly date field as mysql only stores date in the Year-month-day format, whereas the required input is in date/month/year format. After conducting some research, the issue was resolved by inserting the date following the mysql format, while displaying html side format and only convert the date to d/m/Y format when retrieving data to display result.
