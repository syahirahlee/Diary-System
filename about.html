I had issues with storing data into database, particularly date field as mysql only stores date in the Year-month-day format, whereas the required input is in date/month/year format. After conducting some research, the issue was resolved by inserting the date following the mysql format, while displaying html side format and only convert the date to d/m/Y format when retrieving data to display result.
